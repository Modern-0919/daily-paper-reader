## 摘要
大型语言模型智能体在部分可观测、长周期环境中运行，获取监督信号仍是一大瓶颈。我们通过利用现有后训练方法中忽视的监督源来解决这一问题：智能体展开中蕴含的意外但成功的目标。具体而言，我们引入事后监督学习（Hindsight Supervised Learning, HSL），其中辅助LLM审查每个完成的轨迹，并用智能体实际实现的所有自然语言目标对其进行重新标注。然后HSL将轨迹与重新标注的目标配对，并利用这些配对进行额外的微调。为了缓解重新标注数据中的次优性，我们提出了两种HSL学习技术：无关动作掩蔽和样本重新加权。实验表明，HSL灵活且兼容现有的后训练流程。它同时改进了SFT和DPO，在目标空间更多样化的长周期任务上取得了更大的收益。此外，HSL样本效率高：在ALFWorld上，它仅使用四分之一的地面真实演示就超越了在全数据集上训练的基线模型。

## Abstract
Large language model agents operate in partially observable, long-horizon settings where obtaining supervision remains a major bottleneck. We address this by utilizing a source of supervision overlooked in existing post-training methods: unintended yet successful goals embedded within agent rollouts. Specifically, we introduce Hindsight Supervised Learning (HSL), where an auxiliary LLM reviews each completed trajectory and relabels it with all of the natural-language goals the agent actually achieved. HSL then pairs the trajectory with its relabeled goals and uses these pairs for additional fine-tuning. To mitigate suboptimality in the relabeled data, we propose two learning techniques for HSL, irrelevant-action masking and sample reweighting. Our experiments show that HSL is flexible and compatible with existing post-training pipelines. It improves both SFT and DPO, with larger gains on long-horizon tasks with more diverse goal spaces. Moreover, HSL is sample-efficient: on ALFWorld, it surpasses baselines trained on the full dataset while using only one quarter of the ground-truth demonstrations.

## 1. 核心问题与整体含义

大型语言模型（LLM）智能体在部分可观测、长视界的交互环境中（如网页浏览、工具使用、机器人控制）面临一个关键瓶颈：获取监督信号（特别是专家演示）成本高昂且难以覆盖多样场景。现有的后训练方法（如监督微调 SFT）依赖昂贵专家数据，而强化学习（如 DPO、PPO）在稀疏奖励和长视界任务中难以高效探索高回报轨迹。本文旨在利用一个被忽视的监督源：智能体在交互过程中虽然未完成原始指令，但可能无意中实现了其他有效目标（例如在寻找蓝色旗时意外到达了红色旗）。通过事后回顾这些“意外成功”，将其转化为额外的训练数据，可以显著提升智能体的学习效率和性能。

## 2. 方法论

**核心思想**：提出**事后监督学习（Hindsight Supervised Learning, HSL）**框架，通过辅助 LLM 对智能体已完成轨迹进行事后重标，提取所有实际达成的自然语言目标，并将其与对应的轨迹片段配对，用于额外的 SFT 微调。

**关键技术细节**：

- **轨迹收集**：在每个训练步，从训练集采样 b 个指令，智能体与环境交互生成轨迹，直到终止或达到最大步数 T。
- **重标过程**（核心）：使用辅助 LLM M（零样本推理）完成两项任务：
  - **目标识别**：审查整条轨迹，推断出智能体实际达成的一个或多个目标（表示为指令 I′），并提取达成该目标的轨迹片段。
  - **动作相关性标注**：对每个达成目标的轨迹片段中的每一步动作，标注其是否与该目标相关（z_t ∈ {0,1}）。
- **学习策略**：
  - 将重标数据存入动态缓冲区 D′（队列大小 N）。
  - 使用 SFT 目标训练，但引入两项改进：
    - **无关动作掩码**：只对标注为相关的动作（z_t=1）计算损失，避免模仿次优行为。
    - **样本重加权**：根据任务难度和解决最优性调整采样权重，鼓励学习更困难任务的高质量轨迹。
  - 最终损失函数为地面真值演示上的 SFT 损失与重标数据 SFT 损失的加权和（权重 λ）。

**理论支撑**：定理 1 推导了智能体策略 π_θ 与最优策略 π* 差异的上界，表明在专家演示和重标演示上同时进行 SFT 可以收紧该上界，且重标演示的优化有助于缩小覆盖差距。

- **关键设计有效**：同时利用成功轨迹中的中间目标、无关动作掩码、样本重加权均贡献了性能提升。

## 7. 优点

- **创新性**：将机器人领域的 HER（事后经验回放）思想引入 LLM 智能体训练，并针对部分可观测、多目标特性做出关键改进。
- **实用性强**：无需修改环境或奖励函数，仅需一个辅助 LLM 进行零样本重标，即可获得额外训练数据。
- **样本效率高**：极大降低了对昂贵专家演示的依赖，适合现实场景中演示难以获取的情况。
- **兼容性强**：可作为 SFT/DPO 等现有流程的附加模块，易于集成。
- **理论完备**：提供了严格的上界分析，指导了掩码和重加权策略的设计。
- **分析深入**：不仅报告性能，还对重标数据分布、质量、各组件贡献进行详细分析，有助于理解方法有效性来源。

## 8. 不足与局限

- **依赖外部 LLM 重标**：当前使用 Llama-3.3-70B 进行零样本重标，未与智能体联合优化，性能受限于该模型的推理能力。若重标模型在有噪声或长轨迹时出错，可能引入噪声。
- **动作相关性二值化**：将动作简单归为“相关/不相关”，忽略了部分相关或次优但仍有用的动作（如分步购买效率低但确实完成了任务）。更细粒度的相关性建模可能进一步受益。
- **重标数据分布偏差**：某些目标类型（如 examine）在重标数据中出现较少，导致智能体对这些类型性能较低。需要更强的探索策略来覆盖罕见目标。
- **仍需要地面真值演示**：尽管量少，但 HSL 仍需部分地面真值数据来维持训练稳定性（如同许多后训练方法）。
- **增加训练计算成本**：每次重标需要调用 70B 模型，且轨迹收集和重标并行进行，总训练时间增加（但推理成本不变）。
- **实验覆盖有限**：仅使用 Llama-3.2-1B 作为智能体模型，未测试更大参数模型；仅涉及三个基准，缺乏对更开放环境（如 GUI agent、机器人）的验证。
- **假设预定义目标空间**：任务中的有效目标集合已知（如 ALFWorld 的模板）。在开放世界环境中，需要额外机制推断目标空间。
- **未考虑目标价值与安全性**：意外达成的目标可能对用户价值低甚至有害（如购买错误商品），未来需引入目标效用评估和安全过滤。
