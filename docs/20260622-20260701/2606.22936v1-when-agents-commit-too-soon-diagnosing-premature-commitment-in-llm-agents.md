## 摘要
长视界LLM智能体可能悄然失败：它们过早地确定对证据的一种解读，然后花费剩余时间来捍卫这种解读。我们称之为过早承诺。最终答案评分无法捕捉这种失败模式，因为它只看到答案，而不考虑过程是否已经坍缩为一条稳定路径。我们将表征承诺定义为在固定推理步骤上跨运行的隐藏状态收敛，并将其用作轨迹一致性的早期诊断。在Llama-3.1-70B运行于HotpotQA上的ReAct中，步骤4的隐藏状态相似性预测了后续行为一致性（r = -0.35，偏相关r = -0.45），具有局部的时间层和层级特征。该信号在Qwen-2.5-72B和Phi-3-14B以及StrategyQA（r = -0.83）上得到复现。它不跟踪正确性：承诺错误和承诺正确的问题在激活相似性上不可区分。这一边界是主张的核心。承诺告诉我们智能体是否已经稳定，而不是是否正确。运行时监控器从隐藏状态检测不一致轨迹，AUROC高达0.97（更严格划分下为0.85–0.88），而提示干预将行为方差降低了28%（相对于令牌匹配的对照组），同时准确性统计上无变化。我们还测试了该信号是否可用于路由自一致性计算；在一个更难的基准上，它仅略有帮助，并且被一个更简单的基于输出的基线所匹配。结果是针对隐藏过程失败的诊断，具有明确的局限性，而非通用的准确性杠杆。

## Abstract
Long-horizon LLM agents can fail quietly: they settle on one reading of the evidence early, then spend the rest of the run defending it. We call this premature commitment. Final-answer scoring misses the failure mode because it sees only the answer, not whether the process has already collapsed to a stable path. We define representational commitment as cross-run hidden-state convergence at a fixed reasoning step, and use it as an early diagnostic of trajectory consistency. On Llama-3.1-70B running ReAct on HotpotQA, step-4 hidden-state similarity predicts downstream behavioral consistency (r = -0.35, partial r = -0.45), with a localized temporal and layer-wise signature. The signal replicates across Qwen-2.5-72B and Phi-3-14B, and on StrategyQA (r = -0.83). It does not track correctness: committed-wrong and committed-correct questions are not separable in activation similarity. That boundary is central to the claim. Commitment tells us whether an agent has settled, not whether it is right. A runtime monitor detects inconsistent trajectories from hidden states at AUROC up to 0.97 (0.85--0.88 under a stricter split), and a prompting intervention cuts behavioral variance by 28% against a token-matched control while leaving accuracy statistically unchanged. We also test whether the signal can route self-consistency compute; on a harder benchmark it helps only modestly and is matched by a simpler output-based baseline. The result is a diagnostic for a hidden process failure, with clear limits rather than a general accuracy lever.

# 论文总结：When Agents Commit Too Soon: Diagnosing Premature Commitment in LLM Agents

## 1. 核心问题与整体含义（研究动机和背景）

- **核心问题**：长程LLM智能体（如ReAct代理）在执行多步推理任务时，存在一种被忽视的失败模式——**过早承诺（premature commitment）**：智能体在推理早期就固定对证据的单一解读，之后的所有步骤只是捍卫这一初始解释，导致轨迹坍缩为一条稳定路径。
- **现有评估的缺陷**：最终答案评分只看到答案，无法反映过程是否已封闭；跨运行一致性虽能检测行为重复性，但无法区分“正确的一致”和“错误的一致”。
- **研究意义**：提出一种新的过程级诊断方法，通过内部隐藏状态来检测智能体是否已“稳定”，而非答案是否正确。这为运行时监测、节能早退、可控性干预等应用提供了基础，但明确其不能直接提升准确率。

## 2. 方法论：核心思想、关键技术细节

- **核心思想**：**表征承诺（representational commitment）**——在同一问题的多次独立运行（n=10，温度T=0.5）中，计算固定推理步骤s的最后一个token的隐藏状态之间的跨运行平均余弦相似度。高相似度表示智能体已收敛于稳定解读（即使错误），低相似度表示解读仍受具体证据路径影响。
- **关键技术细节**：
  - **ReAct步骤定义**：一个“Thought → Action → Observation”三元组为一个步骤；“步骤4”即第四个三元组。
  - **隐藏状态提取**：从Llama-3.1-70B的8×80GB GPU上流水线并行（每GPU 10层），提取步骤s的最后一个token的隐藏状态，选取层ℓ∈{0,8,16,...,80}（共11层）。
  - **激活相似度计算**：对每个问题q，在步骤s、层ℓ，计算所有运行对的余弦相似度均值：Sim = (n choose 2)^{-1} Σ_{i<j} cos(h_{ℓ,i}^{(s)}, h_{ℓ,j}^{(s)})。
  - **目标指标——行为一致性**：使用步数的变异系数（CV，标准差/均值）作为主要度量，辅以动作序列多样性。两者高度相关（r=0.71）。
  - **统计方法**：皮尔逊相关、偏相关（控制难度和准确率）、置换检验、Bonferroni校正、LOOCV AUROC等。

## 3. 实验设计：数据集/场景、benchmark、对比方法

- **主要数据集**：

  - **StrategyQA**：隐式多步推理（是否类），50个问题，10次运行，用于跨基准验证。
  - **MuSiQue**：更难的复杂多跳QA（150问，10次运行），仅用于测试路由自一致性的能力。
- **Benchmark对比**：
  - 表面基线：问题长度、上下文大小、步骤3的思考长度、检索文档重叠度（Jaccard、TF-IDF、搜索查询重叠）。
  - 下游行为指标：准确率、答案一致性、行为CV。
  - 干预基线：filler提示（与承诺提示令牌数匹配但语义中性）。
  - 路由计算基线：固定样本自一致性（Wang et al., 2023）和输出自适应一致性（Aggarwal et al., 2023）。

## 6. 主要结论与发现

1. **激活相似度预测轨迹一致性**：步骤4（层40）的跨运行相似度与行为CV显著负相关（r=-0.35, partial r=-0.45），顶部相似度四分位数的CV比底部低4.1倍（Cohen's d=1.01）。时间上集中于步骤4，层级上集中在中后部（32-80层）。
2. **跨架构/跨基准复现**：Qwen 72B（峰值r=-0.65, 层64）、Phi-3 14B（峰值r=-0.58, 步骤5/层16）、StrategyQA（r=-0.83, 步骤3）均证实信号存在，但峰值位置因任务和架构而异（步骤数匹配链长，层级深度不同）。
3. **信号追踪一致性而非正确性**：承诺正确和承诺错误的问题在激活相似度上不可区分（Llama p=0.30, Qwen p=0.46）。硬题中信号消失（r≈0），因为承诺错误占主导。答案一致性也与相似度无关（r=-0.13, p=0.22）。
4. **运行时监测**：使用步骤4隐藏态特征的逻辑分类器区分一致/不一致轨迹，AUROC达0.97（极端标签）和0.85-0.88（中位数分割），而问题长度基线仅0.52-0.65。运行数降至3次时AUROC仍达0.81。
5. **提示干预**：步骤3添加承诺提示（vs.令牌匹配的填充提示）使行为CV降低28%（d=0.33, p=0.001），但准确率不变。填充提示虽提升激活相似度但增加CV，证明承诺效应非单纯令牌重叠。
6. **路由计算能力有限**：在更难的MuSiQue上，隐藏态信号仅略优于固定样本自一致性，且被输出自适应一致性基线匹配或超越，仅在小样本预算（约2个样本）时略有优势。建议部署路由需要未来工作。
7. **表征承诺方向近似线性**：承诺方向与第一主成分高度对齐（cos=-0.98），在易题上预测性强，但跨模型对齐性弱（Procrustes cos=0.19）。

## 7. 优点

- **新诊断靶点**：首次系统提出“过早承诺”这一过程级失败模式，并建立跨运行隐藏态收敛作为定量指标，区别于传统的准确率或输出一致性。
- **方法论严谨**：采用置换检验、Bonferroni校正、偏相关控制、中介分析等多种统计手段，区分探索性与确认性分析，报告负面结果。
- **跨模型/跨基准验证**：在三种参数量级（14B-72B）和两种推理类型（检索QA、隐式推理）上复现，增强泛化性。
- **实用性探索**：从诊断延伸到运行时监测（AUROC高、早退省计算）和因果干预（提示削减方差），并诚实报告路由计算的局限性（匹配输出基线）。
- **清晰的概念边界**：明确区分“承诺”与“正确”，并展示两者可拆分，防止误用。
- **可视化与几何分析**：t-SNE、方向投影、层/步热图等帮助理解信号结构。

## 8. 不足与局限

- **检索重叠混淆**：自然情况下，检索相同文档会导致隐藏态相似且行为一致。论文仅部分控制（偏相关、TF-IDF控制后信号消失），需未来实验（固定文档、改变推理）才能因果分离。
- **温度与任务多样性有限**：仅使用T=0.5，未探索温度效应；任务只覆盖QA（HotpotQA, StrategyQA, MuSiQue），缺少编程、数学、具身任务等。
- **行为度量粗粒度**：CV和动作多样性是粗代理，可能遗漏更细粒度的一致性模式（如中间推理步骤的语义一致性）。
- **中介分析不一致（suppression）**：提示干预的中介效应显示间接路径显著但直接路径方向相反，非干净完全中介，需进一步机制分析。
- **单层steering效果差**：尝试的激活操控结果混合，表明承诺分布跨层跨步，需要多层多步操控。
- **等价检验缺失**：承诺正确与承诺错误不可区分的结论仅基于非显著差异，而非严格TOST等效检验，因此只能作为建议性证据。
- **模型全部为指令微调版本**：是否适用于基座模型、长链推理模型等未知。
- **路由计算实用价值不足**：在更难场景下未超越简单输出方法，限制了部署价值。
- **伦理性**：未讨论预防/缓解过早承诺的安全含义（如导致有偏决策的系统性风险）。
